# DataTypes

The data types of Java look as below

```java
/* Numberical Types */

// Mainly used
int aNumber = 42;								// -2,147,483,648 to 2,147,483,648
float aFloatingNumber = 13.37f; // 7 decimal point precision
double aDoubleNumber = 420.005; // 15 decimal point precision

// Sometimes used
byte aTinyNumber = 100;					// (One byte) -128 to 127
short aSmallNumber = 30000;			// -32,768 to 32,767
long aLongNumber = 5000000;			// -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807

/* Booleans */

boolean isRunning = true;				// true (1) or false (0) values
boolean isFun			= false;

/* Characters & strings */
char aSingleCharacter = 'A';		// stores a single character. a-Z 1-0 !-)

String message = "Hello World"; // a sequence of characters
String test = null;							// null is not 0, but literally empty. Has no value!
```
